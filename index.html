实验

  1. 利用所学 HTML、CSS 和设计原则的知识，将自己的 home.ustc.edu.cn/~username 个人主页建立起来。

    • 独立完成HTML和CSS代码，尽量不要使用 Dreamweaver 等所见即所得编辑软件。
    • 尽量遵从 Web 规范，如使用 DIV+CSS 布局。

    提示：W3school 上有很多资料。想实现一个效果，或者遇到浏览器兼容问题时，除了生涩难懂的 specification，最好的参考就是 Google :)

    时间：2011-11-05 ～ 2011-11-30

  2. 动态网站开发练习
    • 最简单的留言板（没有任何用户体系，随便说话）
    • 最简单的博客（自己登录写文章，其他人对文章匿名评论）
    • 最简单的讨论版（可以注册、登录、发帖、回帖）
    • 最简单的论坛（由多个讨论版构成，可以添加讨论版、为讨论版设置版主等）
    • 最简单的SNS（用户可以上传头像、完善个人资料，最重要的是能够互相 follow、发表状态）
    • 最简单的聊天室（实时的讨论版，要使用 Ajax 技术）

    提示：

    • 网站运行过程中，不应该直接修改代码或数据库，所有操作应该在网页上完成。
    • 注意安全问题，尽量避免 SQL 注入、不安全的 HTML、跨站 JS 攻击 等漏洞。

    通过这些小项目的练习，能够掌握 PHP 语言与 SQL 数据库的编程方法，了解社区网站功能的基本范型，为编写具有一定规模的 Web 社区应用打下基础。

    时间：2011-12-01 ~ 2011-12-31
